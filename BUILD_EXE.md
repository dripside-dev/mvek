# Сборка MVEK в один .exe (Windows)

Цель — получить **один файл** `MVEK.exe`, который запускает игру двойным
кликом, без установки Python и без папки с зависимостями.

> ⚠️ **.exe собирается только на Windows.** PyInstaller не умеет
> кросс-компиляцию, поэтому собрать его на Linux/Mac нельзя — нужно
> запустить сборку именно на винде. Зато сам процесс — в один клик.

## Самый простой путь (рекомендую)

1. Скопируй всю папку проекта на Windows-машину.
2. Поставь Python 3.11–3.13 с [python.org](https://www.python.org/downloads/)
   (при установке отметь галочку **«Add Python to PATH»**).
3. Двойной клик по **`build_exe.bat`**.

Скрипт сам создаст окружение, поставит `pygame-ce`, `numpy`, `pyinstaller`
и соберёт игру. Когда увидишь «ГОТОВО!», забирай файл:

```
dist\MVEK.exe
```

Это и есть единый исполняемый файл. Его можно перенести в любую папку
или отдать другу — Python ему не нужен.

## Вручную (если хочется контролировать процесс)

```bat
py -m venv .venv_build
.venv_build\Scripts\pip install pygame-ce numpy pyinstaller
.venv_build\Scripts\pyinstaller --noconfirm --clean MVEK.spec
```

Готовый файл — `dist\MVEK.exe`.

## Что внутри

- `MVEK.spec` — конфиг сборки: режим `--onefile` (всё в одном exe),
  `console=False` (без чёрного окна консоли). Вшивает все ассеты
  (`mvek/assets/` — иконки, спрайты, меню, музыка `zppp.mp3`) и спрайт
  игрока `платон.png`.
- Сейвы и разметка кнопок (`mvek_save.json`, `menu_layout.json`) **не**
  кладутся внутрь exe — они создаются **рядом с exe** при первом запуске,
  поэтому прогресс сохраняется между запусками, а файл остаётся портативным.

## Мелочи

- **Своя иконка.** Положи файл `mvek.ico` рядом со `MVEK.spec` — он
  автоматически подхватится как иконка exe.
- **Антивирус ругается.** Это ложное срабатывание на «неизвестный»
  PyInstaller-exe без подписи — обычное дело, файл безопасен.
- **Меньше размер.** UPX уже включён в спеке; первый запуск onefile-exe
  чуть медленнее, т.к. распаковывается во временную папку.
- **Звуковые эффекты.** Идут через `numpy` (вшивается). Если соберёшь без
  него — музыка останется, а процедурные SFX просто промолчат.
