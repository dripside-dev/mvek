◈ УСТАНОВКА

Пошаговая инструкция

◈ WINDOWS
01
Скачай и установи Python 3.10+ с официального сайта python.org. При установке отметь «Add to PATH».
02
Скачай архив игры и распакуй в удобную папку (например, C:\Games\mvek).
03
Открой командную строку (Win+R → cmd) и установи Pygame: pip install pygame
04
Перейди в папку игры: cd C:\Games\mvek
05
Запусти игру: python -m mvek — или двойным кликом по PLAY.bat
◈ БЫСТРЫЙ СТАРТ

Если Python уже установлен, просто: pip install pygame и двойной клик по PLAY.bat

◈ macOS
01
Установи Homebrew (если нет): скопируй команду с brew.sh и вставь в Терминал.
02
Установи Python: brew install python
03
Распакуй архив игры в домашнюю папку. Открой Терминал и перейди туда: cd ~/mvek
04
Установи Pygame: pip3 install pygame
05
Запусти: python3 -m mvek
◈ LINUX
01
Python 3.10+ скорее всего уже есть. Проверь: python3 --version
02
Установи зависимости (Ubuntu/Debian): sudo apt install python3-pygame или через pip: pip3 install pygame
03
Распакуй архив и перейди в папку: cd mvek
04
Запусти: python3 -m mvek или python3 main.py
◈ ПОМОЩЬ

Частые проблемы

«python не является внутренней командой»

Python не добавлен в PATH. Переустанови Python и отметь галочку «Add Python to PATH» при установке. Или используй полный путь: C:\Python310\python.exe -m mvek

«No module named 'pygame'»

Pygame не установлен. Запусти: pip install pygame. Если несколько версий Python — попробуй pip3 install pygame

Игра запускается, но экран чёрный

Возможно, конфликт с антивирусом или драйвером видеокарты. Попробуй запустить от имени администратора или обнови драйверы.

Медленная работа / низкий FPS

Закрой другие программы. Игра рассчитана на 60 FPS на средних ПК. Если проблема остаётся — напиши нам через форму обратной связи.
